O projekcie

Anime Wheel to innowacyjna platforma stworzona z myślą o miłośnikach anime, którzy często stają przed dylematem wyboru kolejnej serii do obejrzenia. Nasza misja jest prosta: pomóc ci znaleźć idealne anime dopasowane do twoich preferencji.

Projekt powstał z osobistego doświadczenia - sam często spędzałem długie godziny na przeglądaniu różnych stron w poszukiwaniu ciekawych anime. Postanowiłem stworzyć narzędzie, które ułatwi ten proces innym fanom.

Główne funkcje

  • Wyszukiwarka Anime - Zaawansowana wyszukiwarka pozwalająca znaleźć anime po tytule i gatunku
  • Top Lista - Regularnie aktualizowany ranking najlepszych anime według ocen społeczności
  • Losowa Lista - Unikalna funkcja pokazująca losowo wybrane anime dla niezdecydowanych
  • Integracja z MyAnimeList i Shinden - Łączymy dane z najpopularniejszych platform anime

Technologie

Strona została stworzona przy użyciu nowoczesnych technologii webowych:

  • Frontend:
    • HTML5 - Struktura strony
    • CSS3 - Style i animacje
    • Bootstrap 4 - Responsywny design
    • JavaScript - Dynamiczne funkcjonalności
  • Backend:
    • Node.js - Środowisko serwerowe
    • Express.js - Framework webowy
    • API Integration - MyAnimeList API, Shinden API
